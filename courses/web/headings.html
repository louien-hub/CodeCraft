HTML Headings

Introduction

HTML headings are used to define headings or subheadings on a webpage. Headings are essential for structuring content, providing hierarchy, and improving the overall readability of a page.

Types of Headings

HTML offers six levels of headings, from <h1> to <h6>, where <h1> represents the highest level (main heading) and <h6> represents the lowest level (subheading). 

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<h3>This is a Heading 3</h3>
<h4>This is a Heading 4</h4>
<h5>This is a Heading 5</h5>
<h6>This is a Heading 6</h6>

Example Usage:

<!DOCTYPE html>
<html>

<head>
    <title>HTML Headings Example</title>
</head>

<body>

    <h1>Main Heading</h1>
    <p>This is a paragraph of text.</p>

    <h2>Subheading</h2>
    <p>Another paragraph of text.</p>

    <h3>Sub-subheading</h3>
    <p>Yet another paragraph of text.</p>

</body>

</html>

Best Practices

  • Use headings to structure your content logically.
  • Start with <h1> and use subsequent levels in descending order of importance.
  • Avoid skipping heading levels (e.g., going from <h2> to <h4> without <h3> in between).
  • Do not use headings for styling purposes; use CSS for presentation.

Accessibility Considerations

  • Screen readers and other assistive technologies use headings to navigate and understand the content hierarchy.
  • Ensure that your headings accurately represent the structure of your content.

Summary

In summary, HTML headings provide a way to structure and organize content on a webpage. By using headings appropriately, you can create well-organized, accessible, and visually appealing web pages. Experiment with different heading levels to find the hierarchy that best suits your content.
