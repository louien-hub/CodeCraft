Module: HTML Elements

Overview:

HTML Elements: HTML documents are composed of various elements that define the structure and content of a web page. Elements are represented by tags and may contain attributes. Understanding common HTML elements is crucial for creating well-structured and semantically meaningful web pages.

Learning Objectives:

  1. Recognize and use essential HTML elements.
  2. Understand the purpose and usage of common HTML attributes.
  3. Apply semantic HTML for improved accessibility and search engine optimization.

Lesson 1: Basic HTML Elements

1.1 Headings and Paragraphs:

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>
<p>This is a paragraph.</p>

1.2 Links:

<a href="https://www.example.com">Visit our website</a>

1.3 Images:

<img src="image.jpg" alt="Description of the image">

Lesson 2: Text Formatting Elements

2.1 Bold and Italic:

<p><strong>This text is bold</strong></p>
<p><em>This text is italic</em></p>

2.2 Line Break:

<p>This is the first line.<br>This is the second line.</p>

Lesson 3: Lists

3.1 Unordered List:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

3.2 Ordered List:

<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>
