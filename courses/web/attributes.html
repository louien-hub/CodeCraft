HTML Attributes

What are HTML Attributes?

HTML attributes provide additional information about HTML elements. They are always included in the opening tag and are used to modify the element's behavior or provide extra details. Attributes consist of a name and a value and are written in the form name="value".

Common HTML Attributes

id Attribute

The id attribute provides a unique identifier for an HTML element. It is useful for styling with CSS or targeting elements with JavaScript.

<p id="unique-id">This paragraph has a unique identifier.</p>

class Attribute

The class attribute is used to group HTML elements together. Multiple elements can share the same class, allowing them to be styled similarly with CSS.

<p class="highlight">This paragraph has a special style.</p>

style Attribute

The style attribute allows you to apply inline CSS styles to an element.

<p style="color: blue; font-size: 16px;">This paragraph has inline styles.</p>

src Attribute

The src attribute is commonly used with elements like <img> and <script> to specify the source (URL) of an external file.

<img src="image.jpg" alt="A descriptive image">

alt Attribute

The alt attribute provides alternative text for images. It is displayed if the image cannot be loaded and is essential for accessibility.

<img src="avatar.jpg" alt="User's profile picture">

href Attribute

The href attribute is used with the <a> (anchor) element to specify the URL to which the link points.

<a href="https://www.example.com">Visit our website</a>

target Attribute

The target attribute is used with the <a> element to specify where the linked document will be opened. Common values include _blank for a new tab or window.

<a href="https://www.example.com" target="_blank">Visit our website in a new tab</a>

disabled Attribute

The disabled attribute is used with form elements to disable user interaction.

<button disabled>Click me (disabled)</button>

placeholder Attribute

The placeholder attribute is used with form input elements to provide a hint or example of the expected value.

<input type="text" placeholder="Enter your username">

Using Attributes

To use attributes, simply include them in the opening tag of the HTML element. Remember to follow the name="value" format. 

<tagname attribute_name="attribute_value">Content</tagname>

Attributes play a crucial role in enhancing the functionality and styling of HTML elements. Experiment with these attributes to understand how they impact the appearance and behavior of your web page elements.
