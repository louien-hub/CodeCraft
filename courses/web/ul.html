HTML Lists

Unordered Lists

An unordered list is a list of items where the order of the items does not matter. In HTML, unordered lists are created using the <ul> (unordered list) element, and each list item is defined with the <li> (list item) element.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Example:

<!DOCTYPE html>
<html>

<head>
    <title>Unordered List Example</title>
</head>

<body>

    <h1>My Favorite Fruits</h1>

    <ul>
        <li>Apple</li>
        <li>Banana</li>
        <li>Orange</li>
    </ul>

</body>

</html>

Ordered Lists

An ordered list is a list of items where the order of the items does matter. In HTML, ordered lists are created using the <ol> (ordered list) element, and each list item is defined with the <li> (list item) element.

<ol>
    <li>First Item</li>
    <li>Second Item</li>
    <li>Third Item</li>
</ol>

Example:

<!DOCTYPE html>
<html>

<head>
    <title>Ordered List Example</title>
</head>

<body>

    <h1>Steps to Make a Sandwich</h1>

    <ol>
        <li>Gather ingredients</li>
        <li>Spread condiments on bread</li>
        <li>Layer with desired fillings</li>
        <li>Top with another slice of bread</li>
        <li>Cut and serve</li>
    </ol>

</body>

</html>

Nesting Lists

You can also nest lists within other lists to create a hierarchical structure. For example, you can have an ordered list inside an unordered list or vice versa.

<ul>
    <li>Item 1</li>
    <li>Item 2
        <ol>
            <li>Subitem 2.1</li>
            <li>Subitem 2.2</li>
        </ol>
    </li>
    <li>Item 3</li>
</ul>

Example:

<!DOCTYPE html>
<html>

<head>
    <title>Nested List Example</title>
</head>

<body>

    <h1>Continents and Countries</h1>

    <ul>
        <li>North America
            <ul>
                <li>United States</li>
                <li>Canada</li>
                <li>Mexico</li>
            </ul>
        </li>
        <li>Europe
            <ul>
                <li>Germany</li>
                <li>France</li>
                <li>United Kingdom</li>
            </ul>
        </li>
        <li>Asia
            <ul>
                <li>China</li>
                <li>India</li>
                <li>Japan</li>
            </ul>
        </li>
    </ul>

</body>

</html>

Summary

In summary, HTML provides two types of lists: unordered lists (<ul>) for items with no specific order and ordered lists (<ol>) for items with a specific order. You can use the <li> element to define each list item, and lists can be nested to create a structured hierarchy. Lists are commonly used to organize and present information on web pages.
